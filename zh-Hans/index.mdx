# Pantum SDK 文档

欢迎来到 Pantum SDK 文档站。

中文文档按照团队在落地 SDK 时的实际路径组织：

- 先看接入指南，建立整体认知
- 再通过示例工程跑通一个完整环境
- 进入业务开发后使用 API 文档查字段与返回值
- 排障与版本跟踪交给常见问题与更新日志

这样在第一次接入时路径最短，在日常开发与调试中仍能快速定位 API 细节。
