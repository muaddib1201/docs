---
title: 创建预览作业
description: 打开指定 PDF 文件，创建预览作业，返回作业 ID 及文件元数据（页数、版本、创建日期等）。
openapi: openapi/onesdk-core.yaml POST /pdf/preview
---

<Note>
  创建成功后，通过 `job_id` 和 `page_index` 可调用以下接口：
  - [获取指定页元数据](/zh-Hans/api-reference/pdf/pdf-preview-page-meta) — 获取宽高和旋转角度
  - [获取指定页 BMP](/zh-Hans/api-reference/pdf/pdf-preview-page-bmp) — 渲染为 BMP 图像
  - [获取指定页 RAW](/zh-Hans/api-reference/pdf/pdf-preview-page-raw) — 渲染为 RAW 像素数据

  `page_index` 取值范围 [1, `file_meta.page_count`]。
</Note>
